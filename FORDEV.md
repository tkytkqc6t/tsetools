### USEFUL COMMANDS FOR DEV

### How to create and activate virtual environment
## in MacOS:
- create:<br>
python3 -m venv venv

- activate:<br>
source venv/bin/activate

- start server:<br>
python3 server.py

## in Windows:
- create:<br>
python -m venv venv

- activate:<br>
.\venv\Scripts\Activate.ps1

- start server:<br>
python server.py


## Useful commands:
- Fetch all remote branches to local:<br>
git fetch --all

- Show all branches on local:<br>
git branch -r
